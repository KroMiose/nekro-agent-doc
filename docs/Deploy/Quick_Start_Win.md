# 快速开始
::: warning 警告
此文档仅适用于Windows，其他操作系统请勿使用。
:::
::: info 提示
Windows 11 部署前请打开 Windows 开发者选项 PowerShell 脚本执行策略
:::
::: info 提示
部署前请正确安装 Docker Desktop 并开启 WSL2 功能
:::
::: info 提示
部署前检查 区域设置是否打开 UTF-8 编码，打开请关闭并重启电脑
:::
::: info 提示
请勿使用PowerShell 7及以上版本
:::
::: info 提示
Windows 10 1909 及以下版本不支持 WSL2
:::
::: info 提示
Windows 11 22H2 及以下版本不支持 WSL2
:::
::: info 提示
不支持 Windows 10 以下的系统
:::
::: warning 警告
此文档正在施工 
:::
