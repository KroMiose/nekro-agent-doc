# 文档站
## Nekro Agent 文档站
Nekro Agent 文档站是一个基于 vitepress 的文档站，用于管理 Nekro Agent 的文档。
## 快速开始
1. 克隆本仓库
2. 安装依赖
3. 运行
## 架构

主配置文件
侧边栏也在里面：
`.vitepress\config.mts`

全部CSS样式
`.vitepress\theme\css`

主文档目录
`.\docs`

插件文档目录
`.\docs\extensions_docs`

警告
编写文档时，需要注意以下事项：
1. 文档需要使用 Markdown 语法编写
2. 文档内容需要清晰明了
3. 文档请使用 vitepress 扩展语法编写
4. 插件文档命名需要为 作者名_插件名.md
5. 插件文档格式
